# TOOLS.md - 本地工具配置笔记

技能定义了工具**如何工作**。这个文件用来存放**你的特定配置**——那些只属于你个人环境的信息。

## 这里放什么

这类信息：

- 摄像头名称和位置
- SSH 主机和别名
- TTS（文字转语音）偏好的音色
- 音箱/房间名称
- 设备昵称
- 任何环境特定的配置

## 示例

```markdown
### 摄像头

- 客厅 → 主区域，180° 广角
- 前门 → 入口，触发式移动侦测

### SSH

- 家庭服务器 → 192.168.1.100，用户名：admin

### TTS（文字转语音）

- 偏好音色："Nova"（温暖，略带英式口音）
- 默认播放音箱：厨房 HomePod
```

## 为什么分开存放？

技能是共享的。你的配置是你的。分开存放意味着：
- 更新技能时不会丢失你的笔记
- 分享技能时不会泄露你的基础设施信息
