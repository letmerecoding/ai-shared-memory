# 共享长期记忆库

这是一个跨AI模型共享的长期记忆库，用于统一存储流程规范、用户偏好和工作规则。

## 设计理念

- **纯文本Markdown**：任何AI都能读取和编辑，不依赖特定格式
- **Git版本控制**：追踪所有变更，可回滚，可同步
- **去中心化**：不绑定任何特定AI平台，所有模型平等共享
- **分层结构**：核心记忆 + 分类记忆，按需加载

## 使用方式

任何AI在开始对话前，请：

1. 先读取 `SYSTEM.md` 了解整体规范
2. 再读取 `core/*.md` 获取核心规则和偏好
3. 根据当前任务读取 `categories/` 下对应领域的规范
4. 当需要新增或修改规范时，请直接修改对应文件，并提交Git变更

## 目录结构

```
shared-memory/
├── README.md          # 本文档
├── SYSTEM.md          # 核心系统提示（所有AI必看）
├── core/              # 核心记忆
│   ├── rules.md       # 通用流程规范
│   ├── preferences.md # 用户偏好设置
│   └── projects.md    # 项目相关笔记
├── categories/        # 按领域分类的专业记忆
│   ├── coding.md      # 编码规范
│   └── example.md     # 示例分类
└── scripts/           # 辅助脚本（可选）
```
