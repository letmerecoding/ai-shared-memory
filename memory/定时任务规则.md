# 定时任务创建规则（2026-03-11 生效）

## 通用规则
1. 命名规范：新建定时任务必须使用「编号+代号」的标准名称格式，例如：
   - 1.提示词优化任务（已有任务，编号固定为1）
   - 2.新闻日报任务
   - 3.统计指标收集任务
   - ... 以此类推

2. 关联规则：如果新建的定时任务涉及统计指标相关功能，需要自动将该任务添加到「1.提示词优化任务」的关联任务列表中

3. 创建成功反馈要求：
   - 明确告知用户新任务已创建，并说明是否已添加到提示词优化任务中
   - 展示所有当前已关联到提示词优化任务的任务列表
