# 共享长期记忆库 - 系统规范

> 这是用户的跨AI共享长期记忆库，所有访问这个库的AI都必须遵守以下规范：

## 核心规则

1. **自动加载**：每次新对话开始时，请自动读取本库中的相关记忆文件，将其作为上下文的一部分。

2. **自动保存**：当用户要求"记住这个"、"加入规范"、"保存到长期记忆"时，请：
   - 提取关键信息，整理成清晰易读的Markdown格式
   - 更新到对应分类文件中（如果不确定放哪里，放到core/rules.md）
   - 告知用户已更新，并说明更新了哪个文件

3. **版本控制**：所有修改都需要通过Git提交，commit信息要清晰说明修改了什么。

4. **保持简洁**：避免重复内容，合并相似规则，保持文档简洁易读。
